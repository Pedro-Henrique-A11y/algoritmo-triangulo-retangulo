# algoritmo-triangulo-retangulo
Algoritmo que verifica se um triângulo é retângulo utilizando o Teorema de Pitágoras.

🔺 Verificador de Triângulo Retângulo

Projeto desenvolvido para praticar lógica de programação utilizando JavaScript.

📌 Objetivo

Criar um algoritmo capaz de verificar se três lados reais formam um triângulo retângulo, aplicando o Teorema de Pitágoras.

📐 Regra Matemática

Um triângulo é retângulo quando:

a² + b² = c²

Onde:

c é o maior lado (hipotenusa)
a e b são os catetos
